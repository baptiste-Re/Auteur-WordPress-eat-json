Fonctions :
1) Injection automatique AVANT le contenu : "Publié le (date) par (auteur) : date de mise à jour de l'article (date)".
2) Injection automatique APRÈS le contenu : boîte auteur (nom + bio, avatar optionnel).
3) JSON-LD Article + auteur : FORCÉ par défaut, même si Yoast ou Rank Math sont présents (désactivez-le dans Réglages si besoin).

Réglages : Réglages → Ajouter Auteur
- Lien sur le nom de l’auteur (on/off)
- Afficher l’avatar (on/off)
- Forcer le JSON-LD (on/off) — activé par défaut
- Priorités d’injection haut/bas
